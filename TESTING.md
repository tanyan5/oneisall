# Manual Test Checklist

在 Windows 10+ 上执行。默认全局快捷键见 `src/shared/shortcuts.ts`（可在设置页修改）。

## 开发与启动

- [ ] `npm install` 成功
- [ ] `node scripts/gen-icons.mjs` 可生成 `resources/icon.ico`、`resources/tray/`、`plugins/*/icon.png`
- [ ] `npm run dev` 启动无报错，主窗口显示主页（侧边栏含剪贴板、闪开、演示等工具）

## 主窗口与导航

- [ ] 点击侧边栏工具项 → 右侧预览区显示说明与 launchKeywords chip
- [ ] 预览区点击「打开」→ 进入对应工具沉浸式视图（侧边栏隐藏）
- [ ] 侧边栏底部「设置」→ 进入设置页（沉浸式）
- [ ] 侧边栏品牌「OneIsAll」→ 返回主页
- [ ] 沉浸式视图中按 Esc → 返回主页；主页再按 Esc → 窗口隐藏到托盘
- [ ] 窗口置顶（Pin）按钮可切换置顶状态

## 系统托盘

- [ ] 关闭主窗口后进程仍在，托盘显示 O 环图标
- [ ] 双击托盘 → 唤起快速启动浮层
- [ ] 托盘右键「显示」→ 唤起快速启动浮层
- [ ] 托盘右键「设置」→ 主窗口打开设置页
- [ ] 托盘右键「退出」→ 进程结束

## 全局快捷键（默认）

- [ ] `Ctrl+Shift+Space` → 快速启动浮层
- [ ] `Ctrl+Shift+V` → 主窗口打开剪贴板工具（沉浸式）
- [ ] 在设置页修改快捷键并保存后，新组合生效；冲突时有错误提示

## 快速启动浮层

- [ ] 浮层居中显示，失焦后自动隐藏
- [ ] 搜索框可过滤工具与应用；键盘可导航并回车打开
- [ ] 无近期使用时显示推荐工具与固定关键字引导
- [ ] 有近期使用时以 dock 形式展示；O 环按钮 → 主窗口主页
- [ ] 从浮层选择工具 → 打开对应沉浸式工具并隐藏浮层

## 设置

- [ ] 可查看并修改「打开剪贴板」「打开快速启动」快捷键
- [ ] 保存成功有提示；Esc 从设置页返回主页

## 剪贴板 — 捕获与列表

- [ ] 复制文字 → 列表出现新行，时间与摘要正确
- [ ] 复制 HTML/富文本 → 列表正确归类显示
- [ ] 截图或复制图片 → 列表显示缩略图与 `[图片] 宽×高`
- [ ] 在资源管理器复制文件 → 列表显示文件名或「等 N 项」
- [ ] 连续复制相同内容 → 不重复追加
- [ ] 分类 Tab（全部/文本/图片/文件/收藏）过滤正确
- [ ] 搜索框可关键字过滤

## 剪贴板 — 操作

- [ ] 点击行 → 内容写回系统剪贴板，可在记事本等应用粘贴
- [ ] 图片行点击后可在目标应用粘贴图片
- [ ] 文件行点击后可在支持粘贴的文件目标粘贴
- [ ] 工具栏：置顶、详情、复制、多选、收藏、删除、编辑、另存为、清空历史
- [ ] 列表有焦点时 `↑`/`↓` 切换活动行，`Enter` 复制活动行
- [ ] 工具栏单键快捷键（如 `C` 复制、`F` 收藏）在列表焦点下可用
- [ ] 删除/清空后数据消失；重启应用后仍不存在

## 闪开

- [ ] 从主页或 Launcher 打开闪开工具
- [ ] 可创建模块并添加 `.exe` / `.lnk` 快捷方式
- [ ] 点击应用图标可启动对应程序
- [ ] 背景可在「赛博网格」与「Aurora 流光」间切换
- [ ] 数据写入 `%APPDATA%/OneIsAll/data/shankai.json`

## 演示工具

- [ ] 从主页预览或 Launcher 打开「演示」页面正常

## 持久化

- [ ] 退出应用再启动，剪贴板历史仍在（最多 200 条）
- [ ] 图片缓存位于 `%APPDATA%/OneIsAll/data/images/`
- [ ] 设置与固定关键字位于 `%APPDATA%/OneIsAll/settings.json`

## 构建

- [ ] `npm run build` 成功，输出到 `out/`
- [ ] `npm run dist` 成功，安装包或绿色版输出到 `release/`
